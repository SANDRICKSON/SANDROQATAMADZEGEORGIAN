გამარჯობა! მე ვარ სანდრო ქათამაძე - Fullstack, Mobile და Arduino დეველოპერი, ასევე ნარატივ დიზაინერი. ამ ვებგვერდზე შენ შეგიძლია გაეცნო ჩემს პროექტებს და უფრო მეტი გაიგო ჩემ შესახებ. ჩემი თავგადასავალი პროგრამირების სამყაროში 13 წლის ასაკში დაიწყო, როდესაც დაინტერესდი, თუ როგორ იქმნებოდა ის ვებსაიტები და მობილური აპლიკაციები, რომლებსაც ყოველდღიურად ვიყენებდი. ჩემი პირველი ნაბიჯები მოიცავდა HTML, CSS და JavaScript-ით მარტივი ვებგვერდების შექმნას. თუმცა მალევე მივხვდი, რომ მინდოდა ეს ჩემი პროფესიის ნაწილი გამხდარიყო. სწორედ ამიტომ შევუერთდი თიბისი ბანკსა და ჯეოლაბს, სადაც Frontend და მობაილ დეველოპმენტში გავიღრმავე ცოდნა. ამ გამოცდილებამ გამიხსნა კარი საინტერესო პროექტებისკენ, რომელთა ნაწილი TBC Tech School-სა და Geolab-ს უკავშირდებოდა. შემდეგ დავინტერესდი ჩეთბოტების სამყაროთი და Explorium-ში დავიწყე მათი შექმნა როგორც პერსონალურ ვებსაიტებზე, ასევე AnimaChatbotics პლატფორმისთვის. ამ პროცესმა მნიშვნელოვნად გააფართოვა ჩემი უნარები და ხედვები. ასევე გავიღრმავე ცოდნა კიბერუსაფრთხოებასა და თანამედროვე ტექნოლოგიებში ილიას უნივერსიტეტის კიბერლაბორატორიაში—UniLab-ში. ამჟამად ვსწავლობ ჭიათურის მულტიფუნქციურ ცენტრში: რობოტიკას, IT ქსელებს და ხელოვნურ ინტელექტს - GENAIEDU-ს პროექტის ფარგლებში. 2024 წელს მივიღე მონაწილეობა ევროპის რობოტიკის ჩემპიონატში, რაც ჩემთვის ერთ-ერთი უმნიშვნელოვანესი ეტაპი იყო და დიდი შემართებით ვემზადები რობოტიკის 2025 ჩემპიონატისთვის. ამავდროულად, ვარ სტუდენტი Skillwill-ის ნეო უნივერსიტეტში - Python დეველოპმენტის განხრით.
